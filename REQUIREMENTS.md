Тестовое задание: кликер с лидербордом
Необходимо реализовать простой кликер в виде Telegram Mini App. Пользователь нажимает
кнопку — количество его кликов растёт. Все клики должны сохраняться на бэкенде, а
приложение должно отображать текущие результаты и лидеров.
Основная функциональность
• Приложение быть реализовано в виде Telegram Mini App.
• У пользователя есть основная кнопка, которая увеличивает счётчик кликов.
• Для пользователей должен быть экран доступен экран, который показывает топ-25 лидеров
по количеству кликов. Каждый пользователь должен видеть своё текущее место в рейтинге,
даже если он не входит в топ.
• Бэкенд должен быть защищён от накрутки, хотя бы в примитивной форме.
Требования к реализации
• Предполагайте, что мы делаем высоконагруженный кликер, сравнимый по масштабам с
Hamster Kombat.
• Фронтенд: TypeScript + React.
• Бэкенд: TypeScript + Node.js + MongoDB.
• Код должен быть написан лаконично и просто, без излишней сложности.
• Дизайн — на ваше усмотрение, но ожидается минималистичный и единый стиль.
• Избегайте использования тяжеловесных библиотек и фреймворков.
На что мы обращаем внимание
• Чистота и аккуратность кода.
• Архитектура и структура проекта.
• Простота и понятность решений.
• Минимальный, но продуманный UI.
Результат работы
• Код должен быть залит на github в публичный репозиторий.
• Если возможно, задеплойте своё приложение, чтобы им можно было воспользоваться.
• Если задеплоить приложение по какой-то причине невозможно, то будет достаточно видео
с демонстрацией работы.
